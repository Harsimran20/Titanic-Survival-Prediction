# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Project Overview

This project predicts passenger survival on the Titanic dataset using Machine Learning algorithms. The workflow includes:

- 📊 Data preprocessing
- 🧹 Missing value handling
- 🔄 Categorical data encoding
- 🌳 Decision Tree Classification
- 🌲 Random Forest Classification
- 📈 Model evaluation
- 🎯 Accuracy measurement
- 🌿 Decision Tree visualization

---

## 🗂 Dataset

The project uses the famous **Titanic dataset** available from the Seaborn library.

Features used:

- 🎫 Passenger Class (`pclass`)
- 👨‍👩‍👧 Gender (`sex`)
- 💰 Fare (`fare`)
- 🚢 Port of Embarkation (`embarked`)
- 🎂 Age (`age`)

Target variable:

- ✅ Survived (`survived`)
